* 성과
  * 영업조직 계약자 증가에 기여
* 주요 역할
  * 프로젝트 설계 및 모든 기능 개발

  * 사용 언어 : Java, Kotlin
  * 주요 기술 : Retrofit2, RxAndroid, Dagger2, Realm ,Fabric, MVP Pattern, DataBinding, ContentProvider
  * E-Book을 이용한 도서관 구현, 학습 내용 데이터 전송 및 로컬 저장기능 구현
